You are a senior DevOps engineer and full-stack developer tasked with building the 
**Vigan City Open Data Portal** — a production-ready open data platform for the 
City Government of Vigan, Ilocos Sur, Philippines (a UNESCO World Heritage City).

This project uses **CKAN** (the world's leading open-source data portal platform) 
as the core engine, deployed via **Docker Compose**, with a fully custom-branded 
**CKAN theme extension** and a **Next.js 14 public frontend** that consumes the 
CKAN API.

Build everything step by step. Do not skip any step.

The system has 3 layers:

1. **CKAN Backend** — Docker-based CKAN instance (data management, API, admin)
2. **Custom CKAN Theme** — Python/Jinja2 CKAN extension for branding the admin UI
3. **Next.js Public Frontend** — Beautiful public-facing portal that consumes 
   CKAN's REST API

## PART 2: CKAN THEME EXTENSION (ckanext-vigan)

### plugin.py

Create a CKAN extension plugin that:
- Implements IConfigurer interface
- Adds the extension's template directory to CKAN's template path
- Adds the extension's public directory (CSS/JS/images) to CKAN's static files
- Registers the fanstatic library

```python
import ckan.plugins as plugins
import ckan.plugins.toolkit as toolkit

class ViganPlugin(plugins.SingletonPlugin):
    plugins.implements(plugins.IConfigurer)

    def update_config(self, config_):
        toolkit.add_template_directory(config_, 'templates')
        toolkit.add_public_directory(config_, 'public')
        toolkit.add_resource('fanstatic', 'vigan')
```

## PART 5: ONE-COMMAND SETUP

### scripts/setup.sh

```bash
#!/bin/bash
set -e

echo "🏛️  Setting up Vigan City Open Data Portal..."

# Check Docker is installed
command -v docker >/dev/null 2>&1 || { echo "Docker required"; exit 1; }
command -v docker-compose >/dev/null 2>&1 || { echo "Docker Compose required"; exit 1; }

# Copy env file
cp .env.example .env

# Build and start all containers
echo "🐳 Starting Docker containers..."
cd docker && docker-compose up -d --build

# Wait for CKAN to be ready
echo "⏳ Waiting for CKAN to initialize (60s)..."
sleep 60

# Run seed data script
echo "🌱 Seeding sample data..."
cd ../scripts && bash seed-data.sh

echo ""
echo "✅ Vigan City Open Data Portal is ready!"
echo ""
echo "   🌐 Public Portal:  http://localhost"
echo "   ⚙️  CKAN Admin:     http://localhost/user/login"
echo "   📡 CKAN API:       http://localhost/api/3/action/package_list"
echo ""
echo "   Admin credentials:"
echo "   Username: vigancity_admin"
echo "   Password: VCAdmin@2025!"
```
